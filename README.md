# Jekyll AMP

Amp Generator for Jekyll

## Installation

Add this line to your application's Gemfile:

    gem 'jekyll-amp'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jekyll-amp

## Usage

### Configuration

Edit `_config.yml` and add:

```yml
gems: [jekyll-amp]
```
